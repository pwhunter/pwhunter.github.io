Troubleshooting



Warning: Refreshing the page will lose your work!

If you are having difficulty getting things to work, check each of the following:
     
  • Only the symbols ∅, 𝓤, ᶜ, ∪, ∩, \, ⊕, the letters A-Z and a-z, and ( and ) are permitted
    •  
  • The keys 0-6 are shortcuts for the special symbols
    •  
  • |, &, and ' are shortcut keys for ∪, ∩, and ᶜ respectively 
  • Sets should be indicated with a single letter (or ∅ or 𝓤)
    •  
  • Complement binds tighter than the binary operations (∪, ∩, \, ⊕), so A∪Bᶜ will be interpreted as A ∪ (Bᶜ)
    •  
  • The binary operations (∪, ∩, \, ⊕) can only have two arguments and must be enclosed in (), although outermost parentheses can be omitted.
    •  
  •  Parentheses can be added as needed to disambiguate.  If in doubt, add more parentheses - the prover will remove unnecessary parentheses.
  • Only one rule can be applied at only one place on each line.
    •  
 There may still be bugs, if you are still unable to resolve your issue, please contact me detailing the issue.
